# Optimizing Functions with Natural Algorithms

This benchmark evaluate some natural algorithms for optimization task.

The following algorithms were implemented and tested:
- PSO
- BBPSO
- GA
- CMA-ES

# Functions
- Ackley
- Rosenbrock
- Rastrigin 
- Schwefel
- Griewank
